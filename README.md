## 需要的能力
不同的岗位可能需要的能力不同，下面所需的能力只能作为参考。
### 编程能力
* C++
* Ceres、G2O、GTSAM等优化库
* OpenCV
* Eigen
* CUDA
### 算法能力
* 数据结构与算法
* 特征点提取及其匹配原理
* 多视图几何、矩阵论、非线性优化
* 单目/双目/RGB-D SLAM
* 立体匹配
* KF、EKF滤波
* 多传感器融合
